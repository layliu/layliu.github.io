运算符

  • 值
    • undefined 衍生自 null
  • 隐式数据类型转换
    • d=d-0 强制转换成数字
    • d=d+"" 强制转换成字符串
    • 1+ +"2"+3 = 6 +"2"转换成了数字2
    • !!a 非：可以将其他类型转换为布尔值
  • 一元运算符
    • a = ture; a = -a; a则变成numbler ，值为-1
    • a = "18"; a = +a; a则变成number，值为18
  • 自增 ：无论前加加还是后加加，都会使原变量的值自增1
    • a++ ：后加加，把加的值赋值到a上，但表达式的值还是自增前的值
    • ++a ：前加加，把加的值赋值到a上，但表达式的值是自增后的新值
    • var d = 20; result = d++ ++d +d =64
  • 自减：a-- 表达式是原值，--a表达式是新值
  • 逻辑运算符：非+与+或 !+&&+||
    • && 与运算，像爱情。只要两个数值中，有一个为false那就最终返回false，两个值都为true，才返回true
      • js中的与是短路与，如果第一个值为false就不会看第二个值
    • || 或运算，像亲情。只要两个数值中有true，就返回true
      • js中的或是短路或，如果第一个值为true则不会看第二个值
  • 逻辑运算符：与或，非布尔值情况
    • 对于非布尔值会将其先转换为布尔值，再运算：
    • 与运算：短路运算
      • 如果两个值都为true，则返回后一值
      • 如果两个值有false，则返回靠前的false
    • 或运算：短路运算
      • 如果两个值有true，则返回靠前的true
      • 如果两个值都是false，则返回后一值
  • 赋值运算符：
    • +=： a=a+5 等价于 a+=5，+=之前不能有空格
    • -= *= /= %= 都差不多
  • 关系运算符：<>=，
    • 非数值比较，会先转换为数字再比较
    • 如果符号两侧都是字符串，不会将其转换为数字，而会分别比较字符串中第一位的Unicode编码，如果两个字符串中第一位一样则比较下一位
    • 任何值和NaN比较，都是false
    • JS如何输入unicode编码：\u2620
    • 网页中输入unicode编码：&#9760
  • 相等运算符：
    • ==：判断两个值是否相等。值的类型不同，则自动进行类型转换为相同类型。
      • NaN不和任何值相等，包括其本身。要判断值是否为NaN，要用isNaN()函数
    • ！=：判断不相等。
    • ===：全等判断，和==不一样的是不会自动转换值类型。!==是不全等
  • 条件运算符/三元运算符
    • 语法：条件表达试?语句1:语句2；如果表达式为true，则返回语句1，如果为false则执行语句2
    • 获取a和b中的大值：var max = a>b ? a : b 如果a>b，就把a赋值给max，否则赋值b
    • 获取abc三个中的大值：var max = a >b ? ( a>c ? a : c ) : ( b>c ? b : c ) ; 去掉括号
  • 逗号运算符，：用于分隔多个语句，一般可以在声明多个变量时使用 var a=1, b=2;
  • 运算优先级：可查表，遇到拿不准的加括号。
    • 先乘除，后加减。
    • 与优先级高于或。

流程控制语句

  • 代码块：{} 只有给语句分组的作用，不会隔离，外部可见
  • 条件判断：if语句
    • if...else
    • if...else if...else：一个一个判断条件，一旦有个条件满足了，执行后就结束语句。所有条件都不满足，执行最后的else
    • prompt()函数返回值是string类型的，比较数字大小时，记得转换成数值
  • 条件分支：switch case语句：比较 swtich 和 case 后的表达式，依次进行全等比较
  • 循环语句：不会停止的循环叫死循环
    • 循环三要素：初始值，终止条件，更新表达试
    • while循环：和if很像，先判断再执行，只是可循环。
    • do...while 循环：和上面的相比，执行时会先执行do后的循环体，至少会执行一次。
    • for循环： 提供了专门的位置用来放循环三要素。初始化表达试只执行一次。三要素也可以省略或写在外部（分号不可省略）。
      • 求个位数，直接 i%10
      • for循环求水仙花数 
         for (let i = 100; i <= 999; i++){
        var bai = parseInt(i/100);
        var shi = parseInt((i-bai*100)/10);
        var ge = i%10;
        if(i == bai*bai*bai+shi*shi*shi+ge*ge*ge){
          document.write(i+"<br%3E");
        } 
      }```
      • 判断用户输入的是否为质数
      • 打印99成法表
      • 打印所有质数
    • 循环语句可以加标签label，来识别当前循环
    • 性能提升
      • break，可用于退出swtich或循环语句，只对最近的循环产生影响（除非加标签）
      • continue，可跳过当次循环，只对最近的循环产生影响（除非加标签）
      • console.time("计时器名字") console.timeEnd("计时器名字")
      • Math.sqrt(); 可对一个数进行开方
