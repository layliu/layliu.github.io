JS：DOM常用节点方法与属性

2025/03/25
@BB

常用节点操作

获取节点

// 通过ID获取
document.getElementById('id')

// 通过类名获取（返回HTMLCollection）
document.getElementsByClassName('class')

// 通过标签名获取（返回HTMLCollection）
document.getElementsByTagName('div')

// 通过CSS选择器获取（返回第一个匹配元素）
document.querySelector('.class')

// 通过CSS选择器获取所有匹配元素（返回NodeList）
document.querySelectorAll('div.class')

创建节点

// 创建元素节点
const newDiv = document.createElement('div')

// 创建文本节点
const newText = document.createTextNode('Hello World')

// 创建属性节点（通常直接设置属性）
newDiv.setAttribute('id', 'myDiv')

修改节点

// 修改文本内容
element.textContent = '新内容'
element.innerHTML = '<strong>HTML内容</strong>'

// 修改样式
element.style.color = 'red'
element.style.backgroundColor = '#fff'

// 修改类
element.classList.add('new-class')
element.classList.remove('old-class')
element.classList.toggle('active')

节点关系

// 父节点
element.parentNode

// 子节点
element.childNodes // 所有类型子节点
element.children // 仅元素子节点

// 兄弟节点
element.nextSibling
element.previousSibling
element.nextElementSibling // 仅元素节点
element.previousElementSibling // 仅元素节点

添加/删除节点

// 添加子节点
parentElement.appendChild(newElement)

// 插入节点
parentElement.insertBefore(newElement, referenceElement)

// 替换节点
parentElement.replaceChild(newElement, oldElement)

// 删除节点
parentElement.removeChild(childElement)
// 或现代方式
element.remove()

常用元素属性

  • id - 元素唯一标识

  • className - 元素的class属性

  • classList - 更方便操作class的API

  • tagName - 元素的标签名

  • innerHTML - 元素内的HTML内容

  • textContent - 元素内的文本内容

  • value - 表单元素的值

  • style - 元素的样式对象

  • attributes - 元素的所有属性集合
