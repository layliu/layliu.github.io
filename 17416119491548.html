数组方法列表

2025/03/10
@BB
方法名 修改原数组 方法执行后返回值 函数功能 备注 文档链接
push 是 原数组length值 修改数组 添加数据到数组末尾 链接
pop 是 被删除的元素 修改数组 删除数组末尾最后一个数据 链接
shift 是 被删除的元素 修改数组 删除数组开头第一个元素 链接
unshift 是 原数组的length值 修改数组 给数组开头添加数据 链接
splice 是 被删除的数据组成的一个新数组 修改数组 删除或添加数据到数组中 链接
slice 否 新数组 获取数组中部分数据 赋值数组中的数据 链接
*forEach 否 无 处理数据 遍历，无返回值（undefined） 链接
*map 否 新数组 处理数据 遍历，返回新数组 链接
*some 否 布尔值 判断 只要有检测函数返回真值，返回true 链接
*filter 否 新数组 获取数组中部分数据 返回符合检测函数的数据组成的新数组 链接
*every 否 布尔值 判断 所有数据的检测函数都返回真值，返回true 链接
*reduce 否 累加器中的数据 数据处理并加总 返回累加器中的数据 链接
*reduceRight 否 累加器中的数据 数据处理并加总 返回累加器中的数据 链接
*sort 是 原数组 排序 排序专用 链接
reverse 是 原数组 排序 反向排序 链接
*findIndex 否 索引值 找索引 检测函数返回true，那么返回该数据的索引值 链接
indexOf 否 索引值 找索引 返回第一个匹配数据的索引值（从左向右查） 链接
lastIndexOf 否 索引值 找索引 返回最后一个匹配数据的索引值（从右往左查） 链接
*find 否 元素 找元素 返回检测函数返回真值的数据 链接
includes 否 布尔值 判断 如果传入的数据在数组中有，就返回true 链接
concat 否 新数组 合并数组 原数组合并传入的数据（任何数据）到一个新数组 链接
fill 是 原数组 修改数组 用某个数据覆盖填充数组中的数据 链接
copyWithin 是 原数组 修改数组 用数组中的部分数据覆盖原数组中的指定索引位数据 链接
join 否 字符串 转字符串 链接所有数据为字符串 链接
keys 否 可迭代对象 产生可迭代对象 键值，就是属性名（都是数字） 链接
values 否 可迭代对象 产生可迭代对象 值，数组中的数据 链接
entries 否 可迭代对象 产生可迭代对象 键值对，数组的形式，前面是属性名，后面是值 链接
